# Parts MCP

A Model Context Protocol (MCP) server for sourcing electronic parts with KiCad integration.

## Overview

This project provides an MCP server that enables AI assistants to search for and source electronic components, with direct integration to KiCad for seamless PCB design workflows.

## Features

- Electronic parts search and sourcing
- KiCad integration for component libraries
- MCP server implementation

## Project Structure

- `/kicad` - KiCad integration modules
- `/docs` - Project documentation
